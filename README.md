# amir-chatbot101

## :rocket: Quick Start

1. Clone project
```bash
git clone https://github.com/amirdotcom/amir-chatbot101.git
#or
git clone git@github.com:amirdotcom/amir-chatbot101.git
```

2. Create a virtual environment
```bash
python3 -m venv venv #mac
#or 
python -m venv venvn #windows
```

2. Run the virtual environment
```bash
. venv/bin/activate #mac
#or
. venv/Scripts/activate #windows
```

3. Install all the dependecies
```bash
pip install -r/path/to/requirements.txt
#'/path/to = your file path'
```

4. To close the virtual environment
```bash
deactivate
```
